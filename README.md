

تطبيق ويب متطور مصمم بأسلوب "Mobile-First" لمحاكاة تجربة تطبيقات الجوال الأصلية. يتيح التطبيق للمستخدمين تصفح الشاليهات، الفنادق، والمنشآت مع واجهة مستخدم سلسة ونظام تنقل ذكي.

---

## 🛠 التقنيات المستخدمة (Tech Stack)

* **Framework:** [Vue.js 3](https://vuejs.org/) (Composition API)
* **Build Tool:** [Vite](https://vitejs.dev/) (لأداء فائق وسرعة في التطوير)
* **Icons:** [FontAwesome 6](https://fontawesome.com/) (عبر CDN)
* **Styling:** CSS3 (Grid & Flexbox) لضمان استجابة كاملة مع الشاشات.
* **Animations:** Vue Transitions & CSS Keyframes.
